Processing

In the Fall of 2011, I took an art course at the University of Maryland, College Park and made projects using the language Processing. Some were multi-week projects, other were one week "sketches."

The following are some of my standout pieces.

Circle Cross - 2011

A procedural piece of art, using recurssion to make a unique cross like shape.

Git Repo

Gears - 2011

Another piece using recursion to create rotaing rectangles. Also uses transparency to create overlapping color effects as the gears move.

Git Repo

Fireworks - 2011

Fireworks particle system simulator.

Git Repo

Pong Tribute - 2011

An animated piece inspired by the game Pong.

Git Repo

Bouncing Cube - 2011

First attempt doing 3D animation with Processing.

Git Repo

Bouncing Ball - 2011

Another 3D animated piece. This time using the squash and stretch principal of animation as the ball bounces.

Git Repo

Black Canvas - 2011

A small scoped version of an art canvas program. User selects colors and paints. The RGB values are added up at each pixel (which is why the initial page is black).
